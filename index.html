My Key Projects

AI · Recommender · NLP

🎧 MusicMind — Emotion-Aware Hybrid Recommender

A mood-aware music recommendation system that suggests songs based on how you feel—along with clear explanations for each choice. Built a hybrid recommender combining content-based filtering (TF-IDF, Cosine Similarity) and collaborative filtering (SVD) on a 50K+ song dataset, with a personalization engine grounded in psychological emotion models. Integrated real-time explainability and LLM-generated "why this song" insights using a local RAG pipeline, deployed as an interactive Streamlit application.

XGBoost · SHAP · LLM

🌍 AQI Decision Intelligence System

Predicts air quality, explains pollution drivers, and simulates actions to reduce AQI. Built an XGBoost model (R² = 0.92, RMSE=31.5) with SHAP explainability to identify key contributors. Designed a simulation engine to estimate emission reductions needed for target AQI levels. Integrated LLM-based insights (RAG + local model) for contextual guidance, deployed as an interactive Streamlit app.
